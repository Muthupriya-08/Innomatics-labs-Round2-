# Food Delivery Data Analysis

## Datasets
- orders.csv
- users.json
- restaurants.csv

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## Analysis Performed
- User membership analysis
- Revenue analysis by city
- Rating-based revenue
- Cuisine and restaurant insights

## How to Run
1. Clone the repository
2. Install requirements
3. Open `analysis.ipynb` in Jupyter Notebook
